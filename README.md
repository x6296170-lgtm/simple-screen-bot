# Simple Screen Bot

## Project Description
This project is a screen automation bot built using Tesseract.NET. The bot is designed to automate tasks by capturing the screen and processing the images using the Tesseract OCR engine. It can be used for various automation tasks such as data entry, form filling, and other repetitive tasks that require user interaction with the screen.

## Features
- Screen capture capabilities
- Optical Character Recognition using Tesseract.NET
- Automation of repetitive tasks

## Installation
1. Clone the repository: `git clone https://github.com/x6296170-lgtm/simple-screen-bot`
2. Install the required dependencies.

## Usage
- Run the bot and configure it according to your needs.
- Follow the prompts for automation tasks.
